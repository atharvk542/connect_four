# connect 4

to play: 
1. set up a venv with `python3 -m venv .venv` or however you want to do it and activate with `source .venv/bin/activate`
2. install pygame with `pip install -r requirements.txt`
3. start the game with `python main.py`

change between a MinimaxChooser and a HumanChooser in `player_choosers` in `main.py`
or create a new chooser by inheriting the Chooser base class

there is no input column handling at all so be nice to it
